## 三个小想法实现 + .gitignore

### 1. .gitignore
- 新建 `.gitignore`：排除 `public/`、`db.json`、`node_modules/` 等
- `git rm -r --cached public db.json`：从 git 索引移除（本地文件保留）
- 仓库从此瘦身，commit 历史不再被编译产物污染

### 2. 自动版本号
- 新建 `scripts/cache-bust.js`：Hexo filter
- 机制：`git rev-parse --short HEAD` 拿 7 位短 hash，替换 inject 里所有 `{{cache_bust}}` 占位符
- `_config.butterfly.yml` 里所有版本号统一改成 `?v={{cache_bust}}`
- 好处：再也不用手动维护 `?v=20260621a`

### 3. 主题面板适配主题色
- theme-system.css 里 panel-header/title/close/section-label/theme-name 的硬编码颜色全部改为 CSS 变量
- 切换主题时面板自身也跟随变色

### 4. FAB 加返回顶部
- OPTIONS 加第四项 `{ icon: 'fa-arrow-up', label: '顶部', action: 'scrollTop' }`
- 渲染时区分：跳转的用 `<a>`，特殊动作用 `<button>`
- 点击触发 `window.scrollTo({ top: 0, behavior: 'smooth' })`
