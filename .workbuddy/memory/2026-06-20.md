## 修复主题切换按钮 FOUC 闪烁问题

- 问题：博客页面刚打开时，右上角 Theme 按钮会闪白色一下（FOUC）
- 原因：JS 的 `setupNavButton()` 创建按钮时没设初始颜色，等 `applyTheme()` 执行才有颜色
- 修复：
  - `source/js/theme-system.js`：`setupNavButton()` 创建 `<a>` 时立即设 `style.color = '#333333'`
  - `source/css/theme-system.css`：加 `#nav-theme-trigger { color: #333333 !important; }` CSS 兜底
- 已 `hexo generate` + git push 成功，GitHub Pages 线上已更新（commit c7743e2）

## 修复 footer/分页/悬浮按钮/FollowMe 颜色不适配问题
