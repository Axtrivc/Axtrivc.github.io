## Hero 动画：Glitch 故障风转场 v3

把之前平淡的"卷起 + clip-path"转场替换为 **Glitch 故障风**，效果震撼。

### 实现要点
- **8 条 `.hero-glitch-bar`** 错峰向上飞出（横向偏移 `(i-3.5) * 14px`、纵向 `-95vh - i*9vh`、旋转 `±8deg`）
- **SVG filter `#hero-rgb-split`** 用 feColorMatrix 把红绿蓝三通道分离，红左移 4px、蓝右移 4px、绿原位
- **`.hero-noise-spray`** 全屏噪点纹理，0.4s steps(8) animation 模拟扫描
- **文字破碎**：hero-text-headline skewY(-10deg) + translateY(-32vh)，副标题反向 skew
- **CTA 反向缩放飞出**

### 关键 CSS 技巧
- `mix-blend-mode: screen` 让 glitch bar 和 canvas 颜色叠加发光（无需复制 canvas 视觉）
- `feBlend mode="screen"` 在 SVG filter 内合成三通道
- 用 `--trigger = max(0, calc(var(--p) - 0.12))` 让所有 Glitch bar 在 0.12 才开始动，避免 0% 时就出戏
- `hue-rotate(calc(var(--i) * 22deg))` 给每条 bar 不同的色调，更"故障"

### 关键文件
- `scripts/hero-inject.js`：完整转场 CSS + SVG filter + glitch layer HTML
- `source/hero/river-hero.js?v=5`：含 visibilitychange 处理（之前已修复）

### 测试结果（hexo server localhost:4321）
- 0%：hero 完整可见，文字带 RGB 色差（霓虹灯感）
- 25%：glitch bar 飞出，文字模糊偏移
- 35-45%：撕裂感最强，标题飞散
- 50%：博客内容浮现
- 75%：完全过渡
- 100%：纯净博客主页
